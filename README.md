# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) in Python.

## Structure

- `input/` - Puzzle input files (one per day)
- `solutions/` - Solution files (one per day)
- `tools/readers.py` - Shared utility functions for reading input
- `tools/create_day.py` - Script to create input and solution files for each day
- `tools/update_progress.py` - Script to update progress table

## Running Solutions

```bash
python solutions/01.py
```

## Creating Files for a New Day

```bash
python create_day.py  # Creates files for today
python create_day.py 5  # Creates files for day 5
```
